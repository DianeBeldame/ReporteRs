HTML chunks

Get HTML code from ReporteRs objects.

Usage

Function as.html function works with FlexTable, pot objects and raphael-js plots.

It returns the HTML code of the object. That code can then be reused in others applications like knitr or shiny.

shiny Example 

## server.r
#...
output$our_table_output <- renderUI({
    return( div( HTML( as.html( MyFTable ) ) ) )
  })
output$our_text_output <- renderUI({
    return( div( HTML( as.html( my_text ) ) ) )
  })
#...

If you want to get more examples, I strongly recommend to have a look at Victor Perrier 's Github. Victor wrote nice examples showing usage of ReporteRs inside shiny applications.

knitr example

library(ReporteRs)

MyFTable = FlexTable( data = mtcars[1:15, ]
  , header.cell.props=cellProperties(background.color="#003366")
  , header.text.props=textBold( color="white")
  , add.rownames=TRUE
)
# Zebra striped table
MyFTable = setZebraStyle( MyFTable, odd = "#DDDDDD", even = "#FFFFFF" )

my_text_prop = textProperties(color='#1163A5',
  font.size = 20, font.weight = 'bold', font.family = 'Courier New' )

my_text = pot( value = 'My tailor', format = my_text_prop ) +
  ' is rich. ' +
  pot( value = 'Cats', format = chprop( my_text_prop, color = '#F0A91B') ) +
  ' and ' +
  pot( value = 'dogs', format = chprop( my_text_prop, color = '#D63C3A') )

Function as.html is to be called from a chunk tag:

```{r} 

cat( as.html( MyFTable) )
cat( as.html( my_text ) )

```
