FlexTable and knitr

FlexTable objects can be used in knitr documents when output type is HTML.

cell_p = cellProperties(
  border.left.width = 0, border.right.width = 0, 
  border.bottom.width = 2, border.top.width = 2, 
  padding.bottom = 2, padding.top = 2, 
  padding.left = 2, padding.right = 2 )


library(dplyr)
mysummary <- iris %>% group_by(Species) %>% summarize(
  Sepal.Length = mean(Sepal.Length), 
  Sepal.Width = mean(Sepal.Width), 
  Petal.Length = mean(Petal.Length), 
  Petal.Width = mean(Petal.Width) )

bsdoc objects

With bsdoc objects, you can generate an HTML file

  1. Create a bsdoc object

    Use the function bsdoc to create an r object representing an HTML document.

  2. Send R outputs into that object

    Next, you create the components of your HTML document. You create and format text, then insert it using the addParagraph function. You create and insert tables, perhaps merging cells, changing row and column colors or fonts. And, of course, you add plots. Tables, plots and paragraphs formats can be customised with dedicated functions.

  3. Write the object to a file

    Finally, using writeDoc, you write the object into a file with the suffix .html on your file system. Note that other files will be copied in the directory containing the html file (i.e. the bootstrap css, js directories) if it is a new directory.

Example

library( ReporteRs )

# Creation of mydoc, a mydocx object
mydoc = bsdoc( title = 'my document' )

# add into mydoc first 10 lines of iris
mydoc = addFlexTable( mydoc, vanilla.table(iris[1:10,] ) )

mydoc = addParagraph( mydoc, value = "Hello World!" )

# add a plot into mydoc 
mydoc = addPlot( mydoc, function() barplot( 1:8, col = 1:8 ) )

# write the doc 
writeDoc( mydoc, file = "files/bsdoc/www/bsdoc_simple_example.html" )
